# HW4 Narrative Visualization (DATA 227)

This project converts the HW3 football analysis into a narrative visualization app with Streamlit.

## What the app includes

- A central question about converting attacking pressure into points
- Journalistic section flow (context -> evidence scenes -> synthesis)
- Interactive Altair views (selection, metric toggle, and brushing)
- Data from:
  - `PL-season-2324.csv`
  - `PL-season-2425.csv`

## Run locally

1. Create and activate a Python environment (recommended).
2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Start the app:

   ```bash
   streamlit run app.py
   ```

The app opens in your browser and serves the complete narrative plus interactions in one place.

## Notes

- This repository is prepared for local execution of HW4.
- Deployment is intentionally not included.
